# plex-v3

A Plex-inspired theme for Ubooquity 3.1.0.

## Compatibility

Built against Ubooquity `3.1.0`.

This theme relies on the v3 theme structure and is not intended for Ubooquity v2.

## Install

1. Copy the `plex-v3` folder into your Ubooquity `themes` directory.
2. In the Ubooquity admin page, set the theme to `plex-v3`.
3. Restart Ubooquity.
4. Refresh the browser.

## Notes

- The layout is built around a fixed top bar and left sidebar with the content area scrolling inside the frame.
- The home page includes `Recently Added` rows and a theme-side `Continue Reading` row.
- `Continue Reading` is currently implemented as a theme-side workaround using the existing Ubooquity pages and bookmark API. It does not require an external helper service.
- The theme is currently portable as a standalone theme folder.

## Assets

This theme includes a mix of custom styling plus adapted/reused assets from the existing Ubooquity installation for compatibility and visual consistency.

## Status

This is a working community theme, but it is still evolving and may need adjustment if future Ubooquity versions change theme templates, page structure, or bookmark behavior.
